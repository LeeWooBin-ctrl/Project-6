## 📁 Overview

We applied Vision Transformers (ViT, BEiT, Swin) to the CheXpert dataset to detect 14 chest-related conditions.

## 📊 Model Comparison

| Model | Test Accuracy | Test Loss |
|-------|---------------|-----------|
| ViT   | 0.88          | 0.22      |
| BEiT  | 0.89          | 0.18      |
| Swin  | 0.85          | 0.24      |
